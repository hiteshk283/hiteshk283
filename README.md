# Control Center

Control Center is a secure messaging and command platform foundation built for enterprise-grade extension. It combines a Flutter frontend with a Python FastAPI backend, WebSocket support, and PostgreSQL-ready persistence.

## Architecture

- Frontend: Flutter supporting Android, iOS, and Web
- Backend: Python FastAPI with WebSocket foundation
- Database: PostgreSQL
- Deployment: Docker Compose for local development and AWS ECS Fargate readiness

## Contents

- `frontend/flutter_app/` - Flutter application scaffold
- `backend/` - Python backend scaffold with clean architecture separation
- `infrastructure/` - AWS ECS, CloudWatch, networking, security and IAM documentation structure
- `scripts/` - local and deployment script stubs
- `docs/` - architecture, API, deployment, and diagrams documentation
